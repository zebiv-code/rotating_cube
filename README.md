# Rotating Cube

A translucent rotating cube with colored faces rendered in WebGL. Simple, mesmerizing, zero dependencies.

**[Live Demo](https://zebiv.com/rotating-cube/)**

## Tech

- WebGL 1.0 (no libraries)
- Per-face vertex colors with alpha transparency
- Single HTML file, no build step
